# House Price Prediction 🏠💰

## 📌 About
This project predicts house prices using machine learning based on features like area, number of rooms, location, etc.

## 📂 Files
- house_price_prediction.ipynb → main model
- dataset.csv → dataset used
- report.pdf → project documentation

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## ▶️ How to Run
1. Install required libraries:
   pip install pandas numpy matplotlib scikit-learn

2. Open the notebook:
   house_price_prediction.ipynb

3. Run all cells

## 📊 Features
- Data preprocessing
- Feature selection
- Price prediction using ML model
